# Flutter Timer App

A Flutter application that provides an interval training timer for workout sessions.

## Getting Started

1. Clone the repository
2. Run `flutter pub get` to install dependencies
3. Run `flutter run` to start the app

## Dependencies

- Flutter SDK
- audioplayers: For playing sound effects during timer events

## Build Scripts

- `build_web.bat` - Builds the web application for production using `flutter build web`
- `serve_web.bat` - Starts a local server to test the built application
